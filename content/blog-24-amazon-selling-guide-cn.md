## 第三步：账号注册（重点）

### 注册流程

1. 打开 [sell.amazon.com](https://sell.amazon.com) → 点 "Sign up"
2. 用专用邮箱注册（不要用已有买家账号的邮箱）
3. 填写公司信息（英文或拼音）
4. 填写法人信息 + 上传身份证
5. 填写信用卡信息（用于扣月租）
6. 填写收款账户（Payoneer 等）
7. 完成税务审查（W-8BEN 表格）
8. **视频验证** — 法人出镜，展示身份证和营业执照

### ⚠️ 注册避坑

- **不要用 VPN** — 亚马逊会认为你在隐藏身份，直接封号
- **使用干净的 IP** — 不要用公共 WiFi
- **信息必须真实** — 一旦发现虚假信息，永久封号
- **一次填完** — 中途退出可能导致注册失败
- **视频验证准时参加** — 错过预约 = 重新排队 2-4 周

## 第四步：选品 + 上架

### 选品（简化版）

1. 在 Amazon 搜索你想做的品类
2. 看第一页 Review 数（<500 = 可做）
3. 看价格区间（$15-50 = 利润空间好）
4. 看差评（找产品改进方向）
5. 1688 找供应商，要样品

### Listing 上架清单

一个完整的 Listing 需要：

| 元素 | 要求 |
|---|---|
| 标题 | 150-200 字符，品牌 + 核心词 + 3-5 个特征 |
| 5 个 Bullet Points | 每个 150-200 字符，讲利益不是参数 |
| 产品描述 | 1000-2000 字符，含关键词 |
| 首图 | 纯白底，1000x1000 像素以上 |
| 辅图 2-7 | 场景图、尺寸图、细节图、对比图 |
| 后台搜索词 | 250 字节，补全标题没覆盖的关键词 |
| A+ 内容 | 品牌注册后开启（转化率 +10%） |

## 第五步：FBA 发货

### FBA 流程

1. 在 Seller Central 创建 FBA 发货计划
2. 打印 FBA 标签（每个产品贴一个）
3. 装箱 + 打印外箱标签
4. 联系货代发往亚马逊仓库
5. 追踪物流直到入库

### FBA 费用估算（以蓝牙耳机为例）

| 费用项 | 金额 |
|---|---|
| FBA 配送费 | $3.50-5.00/件 |
| 月度仓储费 | $0.50-1.00/件/月 |
| 长期仓储费（>365 天）| $1.50/件/月（尽量避免） |
| 移除/弃置费 | $1.00-2.00/件 |

### 头程物流选择

| 方式 | 时效 | 成本 | 适合 |
|---|---|---|---|
| 海运 | 25-35 天 | 低 | 大批量补货 |
| 空运 | 7-12 天 | 中 | 紧急补货 |
| 快递（DHL/UPS）| 3-5 天 | 高 | 测款 / 小批量 |
| 铁路 | 20-30 天 | 低-中 | 欧洲线 |

**建议：** 首批测款走空运（快但贵），稳定出单后转海运（慢但省）。

## 第七步：处理出单

### 第一个订单之后

1. **24 小时内确认发货**（FBM）或等待 FBA 自动处理
2. **48 小时内回复买家消息**（回复率影响账号健康）
3. **索评**：用 Amazon 官方的 "Request a Review" 按钮
4. **关注账号健康分**：ODR < 1%，发货延迟率 < 4%

### 退货处理

- FBA 退货由 Amazon 处理（退货率 < 10% 属正常）
- FBM 退货必须在 48 小时内退款
- 退货率过高会被限制销售权限

## 新手常见问题

### 需要注册美国公司吗？
不是必须的。大陆公司营业执照可以注册美国站。但美国公司在品牌注册和税务方面有优势。

### 个人卖家能注册吗？
2026 年新卖家基本都需要公司资质。个人卖家账号功能受限且审核更严格。

### 多久能出第一单？
选品 + Listing 做好：1-2 周。配合广告：当天或第二天。Listing 不好：可能永远不出单。

### 月利润能做到多少？
第 1-3 个月：可能亏损或持平（测款成本）。第 3-6 个月：$500-3,000/月。第 6-12 个月：$3,000-15,000/月。第 2 年+：取决于规模。
