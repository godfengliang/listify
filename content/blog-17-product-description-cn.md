## 模板 1：痛点→解决方案

**适合：** 所有品类

```
你是不是经常遇到 [具体痛点]？

[产品名] 就是为了解决这个问题而生的。

• [功能 1] → 不再 [痛点 1]
• [功能 2] → 轻松 [好处 2]
• [功能 3] → 比以前 [对比]

已帮助 [数量]+ 客户解决问题。
```

## 常见问题

### 一个描述能用在所有平台吗？
不建议。每个平台的算法和买家心理不同。Amazon 要 SEO 关键词，Temu 要简洁直接，TikTok 要口语化。

### 中英文怎么选？
看目标市场。美国用英文，日本用日文。Listify 支持 6 种语言一键切换。

### 描述要多长？
Amazon 1000-2000 字符，Shopify 500-1000 字，Temu 200-500 字符，TikTok Shop 100-300 字符。
