## 跨境卖家的终极选择题

每个新手都会问：

> "我应该先做 Amazon，还是先做自己的 Shopify 独立站？"

## 四个维度的全面对比

### 维度 1：流量

| 平台 | 流量来源 | 月活用户 | 获取流量的难度 |
|---|---|---|---|
| Amazon | 站内搜索 | 3亿+ | ⭐ 最容易（搜了就有） |
| Shopify | Google SEO / 社媒 / 广告 | 看你多会引流 | ⭐⭐⭐⭐⭐ 最难（从零开始） |
| Temu | 站内推荐 + 广告 | 1.5亿+ | ⭐⭐ 容易（平台帮你推） |
| TikTok Shop | 短视频 / 直播 | 15亿+ | ⭐⭐⭐ 中等（内容能力强就行） |

**Amazon = 有人已经在搜你的产品，你只需要出现在搜索结果里。**
**Shopify = 没人知道你的品牌，你必须花钱让每个人知道。**

### 维度 2：利润

| 平台 | 费用 | 净利润空间 |
|---|---|---|
| Amazon | 15% 佣金 + FBA 费 + 广告（15-20%） | 15-30% |
| Shopify | 2.9% 支付手续费 + $29/月 | 30-50% |
| Temu | 平台定价，你只能控制供货价 | 5-15% |
| TikTok Shop | 5-8% 佣金 + 达人佣金（10-15%） | 15-30% |

**Shopify 利润最高（没有平台抽成），但前提是你能搞到流量。**

### 维度 3：控制权

| 平台 | 你能控制的 | 你不能控制的 |
|---|---|---|
| Amazon | Listing、价格、广告 | 账号安全（说封就封）、算法变化 |
| Shopify | **一切** — 设计、价格、用户体验、数据 | 没有（你是老板） |
| Temu | 供货价、产品质量 | 零售价、仓储、物流、退货政策 |
| TikTok Shop | 内容、价格、达人合作 | 平台政策（频繁变化） |

**Shopify 给你的控制权是 Amazon 永远给不了的。但控制权 = 责任。**

### 维度 4：风险

| 平台 | 最大风险 |
|---|---|
| Amazon | 账号被封（没有理由、没有申诉渠道） |
| Shopify | 没有流量 = 没有订单 = 白交月租 |
| Temu | 价格战 → 利润归零 |
| TikTok Shop | 政策突变 → 爆款突然违规下架 |

## 四种卖家画像 → 四种策略

### 画像 1：纯新手（$0-2,000 启动资金）

**推荐：** Amazon 美国站

**为什么：**
- 搜一下 "wireless earbuds" → 3000 万结果 → 这些都是在找产品的人
- 不需要自己引流
- 先用 FBA 测款，有单了再考虑独立站

**3 个月计划：**
1. 第 1 个月：上架 3-5 个产品，跑自动广告
2. 第 2 个月：出单的产品优化 Listing，不出单的下架
3. 第 3 个月：好卖的产品扩展到 Shopify

### 画像 2：有内容/短视频能力

**推荐：** TikTok Shop + Shopify

**为什么：**
- 你会拍视频 = 你有流量能力
- TikTok Shop 是 2026 年增长最快的电商平台
- 视频爆了 → TikTok Shop 出单 → 引导到 Shopify 复购

**3 个月计划：**
1. 第 1 个月：每天发 2-3 条产品视频，测试哪种内容能火
2. 第 2 个月：火了的产品开 TikTok Shop，找达人寄样带货
3. 第 3 个月：把粉丝引导到 Shopify（邮件列表 + 品牌独立站）

### 画像 3：有供应链优势（工厂/1688 关系）

**推荐：** Temu + Amazon

**为什么：**
- 你有成本优势，Temu 最喜欢成本低的供应商
- Temu 负责定价、营销、物流，你只管供货
- 同时用 Amazon 做品牌溢价（利润更高）

**策略：**
- Temu：走量，薄利多销（5-15% 利润）
- Amazon：做品牌，高利润（20-30% 利润）
- 同一批货，两条腿走路

### 画像 4：想做长期品牌

**推荐：** Shopify（主战场）+ Amazon（辅助渠道）

**为什么：**
- 品牌是唯一能长期积累的资产
- Amazon 上的 "品牌" 其实不属于你（账号一封什么都没了）
- Shopify 上的品牌永远是你的

**策略：**
1. **内容驱动：** 写博客（SEO 长尾流量）→ Shopify 产品页
2. **邮件驱动：** 获取邮箱 → 自动邮件序列 → 复购
3. **社媒驱动：** TikTok/Instagram → 品牌认知 → Shopify
4. **辅助：** Amazon 作为"信任背书"（顾客搜到你 Amazon 的 Review 会增加购买信心）

## 最赚钱的组合策略

经过对 100+ 个年销百万美元卖家的研究，最常见的组合：

### 组合 A：Amazon + Shopify（最稳定）
- Amazon 负责获客和首次购买
- Shopify 负责品牌建设和复购
- 用 Amazon FBA Multi-Channel 统一发货
- **适合：** 90% 的卖家

### 组合 B：TikTok Shop + Shopify（最快速增长）
- TikTok 负责病毒传播和冲动消费
- Shopify 承接流量和复购
- 用短视频内容驱动增长
- **适合：** 有内容能力的卖家

### 组合 C：Amazon + Temu（最大覆盖面）
- Amazon 做品牌、卖 $25-50 的价格
- Temu 做量、供货价给平台
- 同一个产品，不同定位
- **适合：** 工厂型卖家

## FAQ

### 我已经在做 Amazon 了，要不要关掉做独立站？
不要关。在 Amazon 稳定的基础上开 Shopify。Amazon 用来跑现金流，Shopify 用来积累品牌资产。

### Shopify 没流量怎么办？
三个来源：1) SEO（写博客，3-6 个月见效）2) 社交媒体（TikTok/Instagram，1-2 周见效）3) Google 广告（当天见效，但要花钱）。新手从 SEO 博客开始。

### Temu 把价格压到那么低还有利润吗？
做 Temu 不要想着单品暴利。Temu 的逻辑是：成本 $3 → 供货 $5 → Temu 卖 $12 → 你赚 $2。靠的是量，不是利润率。

### 到底独立站还是平台，给我一个答案？
**先平台（Amazon），做起来了再做独立站（Shopify）。** 这是成功率最高的路线。
